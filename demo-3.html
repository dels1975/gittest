Javascript per il Browser - Eventi relativi alla pagina e alla UI

  • DOMContentLoaded: la pagina è completamente caricata e il DOM costruito
  • click: click con il tasto sinistro del mouse
  • dbclick: doppio click del mouse
  • contextmenu: click con il tasto destro del mouse
  • keydown: pressione di un tasto
  • keyup: rilascio di un tasto
  • cut/copy/paste: taglia/copia/incolla di un testo

Javascript per il Browser - Eventi utili per i form

  • DOMContentLoaded: la pagina è completamente caricata e il DOM costruito
  • focus: il campo acquisisce il focus
  • blur: il campo perde il focus
  • change: il valore di un campo cambia (termina di cambiare)
  • input: il campo riceve un input
  • submit: un form viene inviato
